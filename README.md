# The Early History of the Accademia di San Luca – Resources Canopy IIIF

This repository uses the Canopy IIIF template to build and customize a website of resources for The Early History of the Accademia di San Luca, a project of the Center for Advanced Study in the Visual Arts, National Gallery of Art: https://www.nga.gov/research/center/early-history-accademia-di-san-luca

Documentation, examples, and release notes for Canopy IIIF are available from the main project: https://canopy-iiif.github.io/app/.

## How to use these files

There are several ways to use our project files.

**Explore and search** our site to find documents, images, and transcriptions: https://nationalgalleryofart.github.io/accademia-research/  

**Take the IIIF manifest/collection URL** for the items you are interested to study and import that manifest in a IIIF-compatible viewer or IIIF service. The three main collections are as follows:

- Notarial documents: https://nationalgalleryofart.github.io/accademia-documents-iiif/index.json  
- Maps: https://nationalgalleryofart.github.io/accademia-maps-iiif/index.json  
- Guidebooks: https://nationalgalleryofart.github.io/accademia-guidebooks-iiif/index.json  

**Download the IIIF JSON and/or image tiles** for computational study (e.g., OCR, image analysis, network analysis of names/dates). You may want to use the transcription conventions document to normalize the text before text analysis (e.g., NER, topic modeling): https://nationalgalleryofart.github.io/accademia-research/about/transcriptionsconventions.html  

Please cite items according to the project’s guidance and the rights/statements attached to the original archival items.
